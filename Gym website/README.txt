## Introduction

This web application is designed to provide fitness-related information, including exercise videos and a BMI,Waist-to-Hip Ratio, and Heart Rate Calculator . The project is developed using HTML, CSS, Ajax, Json, and JavaScript.

---

## Features

1. **Exercise Videos:**
   - Burpees
   - Push-ups
   - Mountain Climbers

2. **BMI Calculator:**
   - Input age, gender, weight, height, activity level, and goal
   - Calculates BMI and displays the result

3. **Waist-to-Hip Ratio Calculator
      -Input Waist and Hip
      -Calculates Waist-to-Hip Ratio and displays the result

4. **Heart Rate Calculator
      -input age
      -Calculates the maximum heart rate with your age then displays the result

5. **Terms and Service:**
   - Ability to load and display terms and services dynamically using Ajax

## Usage

To use the fitness website, open the `index.html` file in a web browser. 
Navigate through the various sections, watch exercise videos, and use the BMI calculator.

### Exercise Videos

- Click on the view example of the exercise name to watch the corresponding video.

### BMI Calculator

- Fill in the required information in the BMI calculator section.
- Click the "Calculate BMI" button to see the result.

### Waist-to-Hip Ratio Calculator
   -Fill in the required Information in the Waist-to-Hip Ratio section 
   -Click The "Calculate Ratio" button to see the result


### Heart Rate Calculator
   -fill in the required Information in the Heart rate calculator
   -Click the "Calculate Heart Rate" button to see the result


### Terms and Service

- Click on the "Terms and Service" button to load and view the terms dynamically.

## File Structure

- **index.html:** Main HTML file.
- **about.html:**  About section HTML file
- **contact.html:** contact us section HTML file
- **classes.html** Classes section HTML file
- **nutrition.html:** Nutrition page section HTML file
- **script.js:** JavaScript file containing functions for video display and BMI calculation.
- **styles.css:** CSS file for styling the web application.
- **media/:** Directory containing exercise videos.
- **terms_and_service.txt.txt:** Text file containing terms and service information.

## Styling

The application has a responsive design and uses CSS for styling. Key styling features include:

- Dark background with linear gradient.
- Custom banners for pages
- Interactive Map
- tables with colors every odd or even rows
- Multiple Forms with greyed backgrounds
- Responsive images and videos.
- Grid layout for certain sections.
- Consistent styling for forms and tables.
